# filo

> A Film Logbook App

필름 촬영을 손쉽게 기록할 수 있는 웹 앱입니다.

https://fi-lo.vercel.app/

---

## 사용 방법

### 1. PWA 설치 방법

#### iOS (Safari)

1. Safari에서 https://fi-lo.vercel.app/ 접속
2. 하단 공유 버튼(⬆️) 탭
3. **"홈 화면에 추가"** 선택
4. 이름 확인 후 **추가** 탭
5. 홈 화면의 **filo** 아이콘으로 앱 실행

#### Android (Chrome)

1. Chrome에서 https://fi-lo.vercel.app/ 접속
2. 주소창 오른쪽 **⋮** 메뉴 탭
3. **"앱 설치"** 또는 **"홈 화면에 추가"** 선택
4. **설치** 탭
5. 홈 화면 또는 앱 서랍의 **filo** 아이콘으로 앱 실행

---

### 2. 화면별 사용 방법

#### 기본 데이터 등록

촬영 시작 전에 사용하는 필름, 카메라, 렌즈를 미리 등록합니다.

1. 롤 목록 화면 우상단 기본 데이터 아이콘 클릭
2. 상단 탭에서 **필름 / 카메라 / 렌즈** 선택
3. 이름(필름은 ISO 포함)을 입력하고 **추가** 버튼 탭
4. 등록된 항목은 탭 → 수정 아이콘으로 이름 편집, 삭제 아이콘으로 삭제 가능

---

#### 롤 목록 화면 (`/rolls`)

모든 롤을 확인하고 새 롤을 시작하는 메인 화면입니다.

| 요소                  | 설명                                  |
| --------------------- | ------------------------------------- |
| **촬영중** 섹션       | 현재 활성 롤 표시                     |
| **촬영완료** 섹션     | 마무리된 롤 목록                      |
| **+ 새 롤 시작** 버튼 | 필름·카메라·컷 수를 선택해 새 롤 생성 |
| 롤 카드 탭            | 해당 롤 상세 화면으로 이동            |
| **내보내기** 아이콘   | 전체 데이터를 JSON 파일로 백업        |
| **가져오기** 아이콘   | 백업한 JSON 파일을 불러와 복원        |

---

#### 촬영 화면 (`/shoot`)

롤이 활성 상태일 때 사용하는 실시간 촬영 기록 화면입니다.

| 요소           | 설명                                           |
| -------------- | ---------------------------------------------- |
| 중앙 숫자      | 다음 촬영 컷 번호 / 전체 컷 수 (예: `03 / 36`) |
| **촬영** 버튼  | 탭하면 현재 컷이 기록되고 번호가 증가          |
| 렌즈 표시 버튼 | 탭하면 렌즈 교환 모달 표시                     |
| 진행 바        | 롤 전체 프레임 대비 촬영 진행률                |
| **↩ 되돌리기** | 마지막으로 기록한 컷을 삭제                    |
| **롤 마무리**  | 현재 롤을 완료 상태로 전환                     |
| 리스트 아이콘  | 현재 롤 상세 화면으로 이동                     |

---

#### 롤 상세 화면 (`/rolls/:id`)

특정 롤의 촬영 기록을 확인하고 각 컷 정보를 수정하는 화면입니다.

| 요소              | 설명                                                 |
| ----------------- | ---------------------------------------------------- |
| 롤 상태 뱃지      | 촬영중 / 촬영완료 표시                               |
| 롤 편집 (⚙)       | 필름·카메라·최대 프레임 수·메모 수정                 |
| 프레임 목록       | 기록된 컷 번호, 시간, 렌즈, f값, 셔터속도, 메모 표시 |
| 프레임 탭         | 편집 모달에서 조리개·셔터속도·렌즈·시간·메모 수정    |
| **+ 프레임 추가** | 원하는 위치에 컷 삽입                                |
| **촬영 계속하기** | 활성 롤일 때 촬영 화면으로 이동                      |
| **촬영 재개하기** | 완료 롤을 다시 활성 상태로 전환                      |
| 삭제 (🗑)         | 롤 전체 삭제 (확인 필요)                             |

---

#### 데이터 백업 및 복원

- **내보내기:** 롤 목록 화면 우상단 내보내기 아이콘 탭 → JSON 파일 저장
- **가져오기:** 롤 목록 화면 우상단 가져오기 아이콘 탭 → 저장한 JSON 파일 선택
